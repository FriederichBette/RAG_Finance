# RAG_Finance
Finance RAG Bot with GPT-2
Dieses Projekt demonstriert ein lokales Retrieval-Augmented-Generation (RAG) System mit GPT-2, das Finanzdaten aus einer kleinen, fest kodierten Unternehmensdatenbank abruft und Antworten auf Fragen wie „Welches Unternehmen hat den höchsten Umsatz?“ liefert.

Features:

Feste Unternehmensdaten (Revenue, Profit, Growth)

FAISS für schnelles Retrieval relevanter Dokumente

GPT-2-medium zur Generierung kurzer, konservativer Antworten

Interaktive Abfrage im Terminal

Komplett lokal ausführbar nach einmaligem Modell-Download
